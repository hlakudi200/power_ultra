Open Graph Image for Power Ultra Gym
=====================================

The file 'og-image.png' should be placed in this directory.

Recommended specifications:
- Dimensions: 1200 x 630 pixels
- Format: PNG or JPG
- Max file size: 8 MB (but keep it under 300 KB for performance)
- Content: Power Ultra Gym logo, tagline, or gym photo

This image will be displayed when sharing links to your site on:
- Facebook
- Twitter
- LinkedIn
- WhatsApp
- Slack
- Discord
- And other social platforms

Until you add your custom image, social shares will not show a preview image.

To add your image:
1. Create or design an image with the specifications above
2. Save it as 'og-image.png' in this public/ directory
3. The meta tags in index.html are already configured to use it
